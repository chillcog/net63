# Dark Web Best Practices

Today we will learn about best practices for staying safe on the dark web.

### Explanation
The dark web is risky; follow these to minimize dangers.

#### Key Concepts:
##### Trust: Verify everything.
##### Exit Strategy: Plan to leave safely.
##### Legal Awareness: Know laws.

### Technical:
#### Common Risks:
Malware, scams, law enforcement.

#### Mitigation:
Use VMs, avoid downloads.

## How to setup one properly:

### General Best Practices
1. Don't trust anyone.
2. Use VMs for isolation.
3. Have an exit plan.
4. Stay legal.

### Client (Safe Browsing)
Implement safety measures.

#### Linux/macOS
1. Use Tails for sessions.
2. Enable firewall.
3. Scan for malware with ClamAV.

#### Windows
1. Use Sandboxie for isolation.
2. Install antivirus.
3. Use Tor Browser.

**Note:** Backup nothing sensitive.
